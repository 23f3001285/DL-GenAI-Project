# Messy Mashup Genre Classification

This project builds a deep learning pipeline for music genre classification using synthetic mashups.

Models implemented:
- CRNN (CNN + LSTM) built from scratch
- EfficientNet-B0 pretrained CNN
- AST (Audio Spectrogram Transformer)

Pipeline:
1. Synthetic dataset generation from stems
2. Mel spectrogram feature extraction
3. Model training
4. Ensemble inference with TTA

Final Kaggle score: 0.838
