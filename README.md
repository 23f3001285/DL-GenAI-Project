# DL-GenAI-Project
# 🎧 Audio Genre Classification using Deep Learning

This project focuses on classifying audio mashups into multiple music genres using deep learning techniques. The pipeline explores feature-based and transformer-based approaches, combining them through an ensemble for improved performance.

---

## 📌 Problem Statement

Given audio clips (mashups), the task is to classify each clip into one of several predefined music genres. The challenge lies in handling noisy, mixed signals and extracting meaningful representations from audio data.

---

## ⚙️ Approach

The project follows a structured pipeline:

1. **Exploratory Data Analysis (EDA)**

   * Analyzed audio distributions and waveform characteristics
   * Visualized Mel spectrograms for pattern understanding

2. **Feature Extraction**

   * Converted raw audio into Mel spectrograms
   * Applied amplitude-to-decibel transformation
   * Standardized sampling rate and preprocessing pipeline

3. **Modeling**

   ### 🔹 CRNN (CNN + LSTM)

   * Extracts spatial features using CNN
   * Captures temporal dependencies using LSTM
   * Uses attention-based pooling for better representation

   ### 🔹 EfficientNet (Transfer Learning)

   * Pretrained EfficientNet model adapted for spectrogram input
   * Leverages learned visual features for audio classification

   ### 🔹 AST (Audio Spectrogram Transformer)

   * Transformer-based architecture operating on raw waveform
   * Uses pretrained weights and fine-tuning

4. **Ensemble Strategy**

   * Combined predictions from all models
   * Weighted averaging based on validation performance
   * Improved overall robustness and accuracy

## 🚀 How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Feature Extraction

```
python src/extract_features.py
```

### 3. Train Models

```
python src/train_crnn.py
python src/train_effnet.py
python src/train_ast.py
```

### 4. Inference

```
python src/inference.py
```
